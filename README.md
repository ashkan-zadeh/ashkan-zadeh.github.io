# Ashkan Yousefi Zadeh Portfolio

Static portfolio website for GitHub Pages.

## Files

- `index.html` - main page
- `css/styles.css` - visual design and responsive layout
- `js/main.js` - mobile navigation, active section highlighting, and reveal animation
- `asset/img/Ashkan_CV_Feb_2026.pdf` - downloadable CV
- `asset/img/ashkan-profile.jpg` - optimized profile image used by the page
- `asset/img/xai-av-hero.png` - hero visual

## GitHub Pages

Upload the full folder to a GitHub repository. In repository settings, enable GitHub Pages from the main branch root. No build step is required.
